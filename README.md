Description:
ghad-afdal is a web application developed for the organization Ghad Afdal. The platform helps the organization manage users efficiently, visualize data, and provide real-time notifications. It also includes a dashboard for a quick overview of key metrics and activities.

Key Features:

User Registration & Management: Secure registration and user profile management.

Data Visualization: Graphs and charts to display insights and metrics.

Real-Time Notifications: Instant updates for important events or changes.

Dashboard: Centralized dashboard to monitor statistics, recent activity, and key performance indicators (KPIs).

Responsive Interface: Built with Django templates for a smooth user experience.
Technologies Used:

Backend: Python, Django

Frontend: Django templates (HTML, CSS, JavaScript)

Database: MySQL / PostgreSQL

Libraries: See requirements.txt for all dependencies
